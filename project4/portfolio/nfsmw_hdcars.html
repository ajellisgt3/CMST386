HD Car Model Overhaul - Need for Speed: Most Wanted

The goal of this project is to update all the car models in Need for Speed: Most Wanted in order to improve visual style when playing on modern PCs.

The new models have a higher poly count, use 3D doorseams instead of 2D doorline textures, have more detailed features, and most bodykits have been redesigned or replaced to appear more modern. All of these models take components from various other models. Many bodykits were modeled from scratch, including the first Golf GTI kit and the Mustang bodykit, which is intended to be a modern interpretation of one of the original bodykits from the game. The WIP Porsche 996 bodykit shown is an example of "HD-ifying" an existing kit from the game.
