Porsche Carrera GT Model Fix

This project's aim was to redo the Porsche Carrera GT model in several Need for Speed games, where the original model has been known to be very inaccurate.

For those unaware, EA/Blackbox kept using the same inaccurate Carrera GT model from Need for Speed: Most Wanted all the way through Undercover (and some non-mainline games as well). The inaccurate model apparently was modeled after the concept version of the car and then made to look like the production version, thus resulting in the modelâ€™s odd and inaccurate shape.

This project fixes the Carrera GT in Need for Speed: Most Wanted and Need for Speed: Carbon, remaining faithful to the original art style and textures, and preserving all original customization options, re-adapting body kits and bumpers to the fixed model.
